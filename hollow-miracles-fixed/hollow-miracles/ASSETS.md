# Ассеты Hollow Miracles

Все файлы лежат в `public/assets/`. Игра не падает без любого из них — включается фолбэк (силуэт, процедурный пол, тишина). Хочешь заменить арт — просто положи файл с тем же именем.

## Спрайты бойцов — `assets/anims/`

По 3 кадра на бойца, тёмный фон превращается в прозрачность автоматически (luma-alpha). Ключи: `veil`, `rabbit`, `kain`, `sectarian`, `duk`, `king`.

```
<id>_idle.png   — боевая стойка (смотрит ВПРАВО)
<id>_atk.png    — кадр удара 1
<id>_atk2.png   — кадр удара 2 (опционально)
<id>_atk3.png   — кадр удара 3 (опционально)
<id>_dash.png   — кадр рывка (смер позади, фигура справа)
```

Кадры удара чередуются по тактам: каждый новый замах — следующий кадр. Есть только `_atk` — игра спокойно работает с одним.

Рекомендуемый размер: до 560 px по большей стороне. Фолбэк: старый `assets/<id>.jpg` на все позы.

## Портреты — `assets/portraits/`

```
<id>.png        — аватарка для HUD в бою (вертикальный кадр, лицо сверху)
<id>_card.png   — карточка выбора порядка (в полный бюст)
```

Фолбэк: `assets/<id>_p.jpg` и `assets/<id>.jpg`.

## Карты — `assets/maps/`

```
void.jpg · temple.jpg · forest.jpg   — фон арены, рисуется на всю сцену 1200×700
```

Лежат HQ-арены вида сверху 1400×817 (фиолетовый круг · храм · гнилой лес). Захочешь чётче — сгенерируй новые в этом размере (или крупнее с тем же соотношением 12:7) и перезапиши. Геометрия препятствий — в `server/balance.js` → `MAPS`.

## UI — `assets/ui/`

```
menu_bg.jpg     — фон главного меню (Кролик слева, Король справа уже в кадре)
victory_bg.jpg  — экран победы (надпись ПОБЕДА в самом арте)
defeat_bg.jpg   — экран поражения
```

## Аудио — `assets/audio/` (опционально)

```
theme.mp3    — главное меню и лобби (луп)         ← Black_Door_Lobby
battle.mp3   — бой, вариант 1 (луп)               ← Shadowed_Crest
battle2.mp3  — бой, вариант 2 (луп, чередуются)   ← Heart_in_Ashes
victory.mp3  — стингер победы                     ← Chalice_of_Crows
defeat.mp3   — стингер поражения                  ← Ashen_Game_Over
```

Не нравится раскладка — просто переименуй файлы. Нет файла → игра молчит без ошибок. Кнопка ♪ управляет всем звуком, состояние запоминается.
