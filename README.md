# † HOLLOW MIRACLES — Арена Пустоты

*Faith. Blood. Wonder.*

Онлайн-файтинг 3v3 для двоих: **«Орден Пустоты»** (Вейл, Квазарный Кролик, Каин) против **«Культа Бегемота»** (Сектант, Святой Дюк, Грибной Король). Реалтайм-бой в браузере, с разных устройств, по коду комнаты. Тёмное соулс-фэнтези: три арены, фирменный арт, музыка.

## Запуск локально

Нужен Node.js 18+.

```bash
npm install
npm start
```

Открой `http://localhost:3000` — готово.

## Как сыграть вдвоём

1. **Игрок 1** жмёт «Открыть врата», выбирает орден → получает 6-значный код. Как хост он же выбирает арену.
2. **Игрок 2** вводит код в «Присоединиться» → попадает в комнату.
3. Оба расставляют порядок трёх бойцов (или жмут «🎲 случайный порядок») и жмут «К БОЮ».

Кнопка **«🎲 рандомный бой»** в меню делает всё сама: случайный орден, случайная арена, случайный порядок — остаётся кинуть код другу.

### Вариант А: вы в одной Wi-Fi сети
Игрок 1 запускает сервер и узнаёт свой локальный IP:
- Windows: `ipconfig` → IPv4-адрес (типа `192.168.1.42`)
- macOS/Linux: `ifconfig` или `ip a`

Игрок 2 открывает `http://192.168.1.42:3000` (подставь свой IP).

### Вариант Б: через интернет, быстро (туннель)
```bash
npx localtunnel --port 3000
```
Получишь ссылку вида `https://xxxx.loca.lt` — кидай её другу. Альтернативы: `ngrok http 3000`, Cloudflare Tunnel.

### Вариант В: постоянный хостинг (бесплатно)
Например, [Render](https://render.com): New → Web Service → твой репозиторий. Build: `npm install` · Start: `npm start` · план Free. Подойдёт и Railway / Fly.io / Glitch — порт берётся из `process.env.PORT`. Есть `Dockerfile`, если хостинг хочет контейнер.

> На бесплатном Render сервис засыпает без трафика — первый заход может грузиться ~30 секунд.

## Экраны

Главное меню — три плашки: **ИГРАТЬ** (создать лобби или войти по коду), **ЛОР** (история мира по эпохам) и **ЯРИК** (загляни — узнаешь). В бою сверху-слева есть кнопка **← выйти**: с подтверждением, сопернику засчитывается победа. Из комнаты тоже можно выйти кнопкой внизу.

## Управление

| Действие | Десктоп | Телефон |
|---|---|---|
| Движение | WASD / стрелки | джойстик слева |
| Атака | J / ЛКМ (можно зажимать) | кнопка ⚔ |
| Скилл | K / ПКМ | кнопка ✦ |
| Рывок | Space / Shift | кнопка ➶ |

Телефон лучше держать горизонтально. Кнопка ♪ в правом верхнем углу — звук вкл/выкл.

## Арены

Хост выбирает арену в комнате перед боем:

| Арена | Особенность |
|---|---|
| **Арена Пустоты** | Чистая пустота, без препятствий |
| **Забытый Храм** | Четыре колонны по углам |
| **Гнилой Лес** | Грибные завалы сверху, снизу и по диагонали |

Препятствия считаются на сервере: за них можно прятаться, рывок Короля обрывается о них, как о стену.

## Анимация и эффекты

Бойцы живут на арене: дышат в стойке, подпрыгивают и наклоняются на бегу, оставляют пыль из-под ног и шлейф призраков на рывке. Каждый замах — один из трёх кадров удара с концепт-листов. Павший боец оседает и растворяется.

Скиллы читаются с экрана: **Приговор Луны** Каина поднимает луну со столбом света и рунами на гравикольце, **Квазарное Обнуление** Кролика гасит арену в пустоту с пульсом белых колец, в тумане **Вейла** противник видит лишь розовые искры, ослепление **Сектанта** сжимает обзор багровой кромкой, метка **Дюка** — золотой пульсирующий крест, а попадание бьёт столбом света с неба, в спорах **Короля** прорастают грибы.



## Ростер — 13 бойцов

Каждая фракция собирает родственных по духу персонажей; игрок выбирает **любых троих** из ростера своей стороны.

**Орден Пустоты:** Вейл (ассасин), Квазарный Кролик (маг), Каин (воин), Отражение Бога (анти-контроль), Наследница Видения (снайпер), Люциан Вальмонт (дуэлянт), Спайгер (скоростной).

**Культ Бегемота:** Сектант Бегемота (колдун), Святой Дюк (стрелок), Грибной Король (гигант), Вождь Червей (призыватель), Барогун Гилзкернес (тяжёлый стрелок), Защитник Храмов (танк).

У каждого свой активный скилл и пассивка — от роя личинок Вождя и божественного удара Отражения до парирования Защитника и разгона Спайгера.

## Правила боя

- Формат **3v3 последовательно**: пал боец — выходит следующий. Проиграл тот, у кого пали все трое.
- У каждого бойца: обычная атака (бесплатно), **активный скилл** (мана, кнопка скилла) и **пассивка**.
- Перед ударом противника видно **зону атаки** (сектор или линию прицела) — успей выйти.
- Если игрок отвалился, бой ставится на паузу до 25 секунд — можно перезайти по той же ссылке.
- После боя — кнопка реванша.

## Баланс

Все числа — в `server/balance.js`: формулы статов, параметры скиллов, тайминги, геометрия арен (`MAPS`). Меняй и перезапускай сервер.

## Тесты

```bash
npm test
```

Интеграционный тест поднимает сервер, подключает двух ботов и прогоняет полный матч: комната → старт → движение → атаки → скиллы → свопы 3v3 → клон Короля → победа → реванш → реконнект.

## Структура

```
server/
  index.js    — HTTP + Socket.IO, комнаты, выбор карты, реконнект
  game.js     — боевая симуляция (авторитарный сервер, 30 Гц) + коллизии препятствий
  balance.js  — конфиг персонажей, формулы, карты (MAPS)
public/
  index.html, css/style.css, js/client.js — клиент (Canvas, предсказание+интерполяция)
  assets/     — арт и звук (см. ASSETS.md)
test/sim.js   — интеграционный тест
```

Игра запускается и без части ассетов: любой отсутствующий файл тихо заменяется процедурным фолбэком.
